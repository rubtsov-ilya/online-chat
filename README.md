# Online Chat (React + TypeScript)
A real-time chat application built with React, TypeScript, and modern web technologies.  

## Technologies  
- **Frontend**: React 19, TypeScript, Vite  
- **Styling**: CSS Modules
- **State Management**: Redux Toolkit / React Context  
- **API**: Firebase
